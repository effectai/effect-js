# getPrice

## Description

By calling this function, you will get the current price of the EFX token according to DefiBox.
The price is in USDT. The contract can be viewed at the following link: https://www.bloks.io/account/tethertether
It is not needed for this function to connect a client or a session.


## Usage

```ts
import { getPrice } from '@effectai/sdk';
const price = await getPrice();
console.log(price);
```

## Output

```
0.023399935809187228
```
