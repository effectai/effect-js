# deposit

## Description

User are able to deposit funds into their vAccount or claim the funds from escrow and have them deposited in to their vAccount.
Depositing funds into the vAccount is nessesary to be able to pay for creating campaigns, batches and tasks.

## Parameters

### Client
- **Description:** The client object, **must** be connected with a Session.

### vAccountId
- **Description:** The vAccount id, where the funds should be deposited.


### amount
- **Description:** The amount of funds to be deposited.

## Returns

**Type:**  TransactionResult 

**Description:** 
Returns a transaction response object that contains the transaction id block number, and various properties that correlate to the transaction.
Read more about the transaction response here: [TransactionResponse](/docs/glossary/transaction-result.mdx)
